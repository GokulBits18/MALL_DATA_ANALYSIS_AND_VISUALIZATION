# 🛍️ Mall Customer Data Analysis & Visualization

A web-based data analytics application that analyzes mall customer behavior through interactive dashboards, statistical insights, and machine learning-powered customer segmentation.

## 📌 Overview
This project processes and visualizes mall customer data — including age, gender, annual income, and spending score — to uncover meaningful patterns. It applies **K-Means clustering** to segment customers into distinct groups, enabling data-driven marketing strategies and personalized customer targeting.


## 🚀 Features
- Interactive Dashboard with KPIs
- Customer Segmentation using K-Means clustering
- Data Visualizations (histograms, scatter plots, heatmaps)
- CSV Upload & instant analysis
- Full report generation with marketing recommendations
- User Authentication (Login/Register)
- Admin Module for data and user management

## 🛠️ Tech Stack
- **Backend:** Python 3.7+, Flask
- **Data Processing:** Pandas, NumPy
- **Machine Learning:** Scikit-learn
- **Visualization:** Matplotlib, Seaborn
- **Frontend:** HTML, CSS, Bootstrap 5, Chart.js
- **Dataset:** Mall Customers Dataset (Kaggle)
